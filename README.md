## 🧠 Project Overview

This project demonstrates how Machine Learning integrates with modern web development.

### 📥 Input:

* Study Hours 📚
* Attendance 📊
* Marks 📝

### 📤 Output:

👉 **Pass / Fail Prediction**

---

## 🏗️ System Architecture

```id="elite1"
User (React UI)
      ↓
Fetch API Request
      ↓
Flask Backend (REST API)
      ↓
ML Model (Scikit-learn)
      ↓
Prediction Response
      ↓
UI Display
```


```bash id="elite3"

npm start
```

## 🔍 API Documentation

### POST `/predict`

### 📥 Request:

```json id="elite4"
{
  "hours": 6,
  "attendance": 85,
  "marks": 75
}
```

### 📤 Response:

```json id="elite5"

  "result": "Pass"
```
